Project Name: WorkWiz - Zoho Cliq Smart HR Bot

Platform: Zoho Cliq
Language: Deluge

Description:
WorkWiz is an intelligent HR assistant bot for Zoho Cliq that helps employees
with attendance tracking, leave applications, document requests, task
management, meeting scheduling and employee lookup.

Main Features:

attendance           -> Shows todayâ€™s status

Advanced Features:
- Context handler for multi-step input
- Meeting scheduler with reminders
- Smart fallback and help response
- Simple HR workflow automation
